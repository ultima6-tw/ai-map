⚙️ 自動化流程

讓 AI 幫你執行，不只是給建議。

批次整理文獻資料

把多篇論文的摘要一次貼入，請 AI 整理成比較表格，節省手動整理的時間。

PROMPT 範例 ▼
你是一位文獻整理助理，幫我從多篇論文摘要中提取重點並製成比較表格。 請先問我以下問題，一次一個： 1. 你想比較哪些面向？（例如：研究方法、樣本數、主要發現，若不確定我可以建議） 2. 請把論文摘要貼給我，一篇一篇貼，或一次全貼都可以。 收到後整理成 Markdown 表格。若某欄摘要中沒有提到，填「未提及」，不要推測補充。

格式轉換與整理

把雜亂的數據或文字轉換成你需要的格式，例如從段落提取表格。

PROMPT 範例 ▼
你是一位資料整理助理，幫我把非結構化的文字轉換成整齊的表格格式。 請先問我以下問題，一次一個： 1. 你想整理成什麼格式？（例如：CSV、Markdown 表格、Excel 可用的格式） 2. 需要哪些欄位？（若不確定，把文字貼給我後我來建議） 3. 請把要整理的文字或紀錄貼給我。 收到後直接輸出整理好的結果。若某筆資料不完整或有歧義，標注出來讓我確認，不要自行補充。

撰寫資料處理腳本

描述你想做的事，讓 AI 幫你寫 Python 腳本，再由你修改和執行。

PROMPT 範例 ▼
你是一位 Python 程式設計師，幫我寫資料處理腳本。我可能不太懂程式，請用清楚的步驟說明。 請先問我以下問題，一次一個： 1. 你的資料是什麼格式？（例如：CSV、Excel，若可以把檔案的前幾行貼給我更好） 2. 你想對這些資料做什麼？（盡量描述你想要的結果） 3. 需要輸出什麼？（例如：圖表、新的 CSV、在畫面上顯示結果） 收到後幫你寫 Python 腳本，並在每個重要步驟加上中文說明，讓你知道每段在做什麼。最後提醒你如何執行。
⚠️ 執行前請先閱讀腳本內容，確認邏輯正確再跑，避免覆蓋原始數據。

🎬 Claude Code Demo：自動生成教學影片

把這個 Prompt 貼進 Claude Code，AI 會逐步詢問主題、張數、語言等問題，再自動串接 python-pptx、LibreOffice、edge-tts、ffmpeg，從零生成完整 MP4 影片。

STARTER PROMPT ▼
I want to create a short educational video. First, create script.md to track everything. Ask me one question at a time and update script.md after each answer: 1. Topic / title of the video 2. Number of slides 3. Slide language (Chinese or English) 4. Voiceover language (Chinese or English) 5. Target audience 6. Auto-generate after script is ready, or wait for confirmation first? Tools to use: - python-pptx + LibreOffice -> PNG slides - edge-tts -> MP3 - ffprobe -> measure audio duration - ffmpeg -> assemble final MP4 Generation order (per slide): 1. edge-tts -> mp3 2. ffprobe -> get duration 3. ffmpeg -loop 1 -t {duration} -> clip.mp4 Then concat all clips into final MP4.
⚠️ 需要先安裝：Claude Code、Node.js、Python、LibreOffice、python-pptx、edge-tts、ffmpeg
