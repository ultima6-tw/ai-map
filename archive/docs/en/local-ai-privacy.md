# 🏠 Local AI & Privacy

Why run AI locally? For **data security**, **zero censorship**, and **no subscription fees**. This is also the first step toward achieving **[Digital Sovereignty](../digital-sovereignty/)**.

## Core Tools
- **[Ollama](https://ollama.com/)**: The simplest way to run models like Llama 3, Mistral, and Gemma locally.
- **[LM Studio](https://lmstudio.ai/)**: Provides a GUI to search and download GGUF models from Hugging Face.
- **[Jan.ai](https://jan.ai/)**: Open-source, privacy-focused local AI chat tool supporting all major OS.

## Local Model Recommendation
- **TAIDE-Llama3-8B**: A Llama 3 fine-tuned model optimized for Taiwan's local linguistic habits, ideal for Traditional Chinese document processing.

## Hardware Recommendations (2026 View)
- **[Can I Run AI?](https://www.canirun.ai/)**: Quickly test if your hardware is capable of running mainstream AI models like Llama 3/4 or Stable Diffusion.
- **VRAM is Key**: The size of the model depends on Video RAM. 8GB can run 7B-8B models, while 16GB-24GB (e.g., RTX 3090/4090/5080) can handle 70B models.
- **Mac Users**: Apple Silicon (M2/M3/M4 Max) unified memory architecture is excellent for running large parameter models.
