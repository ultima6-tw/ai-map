# 🕸️ 知識圖譜 (Knowledge Graph)

知識圖譜 (KG) 是超越傳統關鍵字搜尋與單純向量檢索的技術，它透過建立實體 (Entity) 與關係 (Relationship) 的映射，幫助 AI 理解資料的深層語義結構。

## 🧠 為什麼 OpenClaw 需要知識圖譜？

在 OpenClaw 的架構中，單純的向量檢索 (Vector Search) 無法處理複雜的多節點狀態或長期的行為依賴關聯。透過整合知識圖譜：
1. **提升事實準確度**：確保跨會話的知識一致性。
2. **邏輯推理能力**：讓 AI 能夠沿著圖譜的「關係」進行多跳推理 (Multi-hop Reasoning)。
