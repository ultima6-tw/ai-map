LLM Applications

⚙️ Workflow Automation

Let AI execute tasks for you — not just give advice.

Batch Literature Organization

Paste multiple paper abstracts at once and ask AI to organize them into a comparison table, saving hours of manual work.

PROMPT EXAMPLE ▼
Below are the abstracts of five papers. Please organize them into a comparison table with the following columns: Paper Title, Research Method, Sample Size, Key Findings, Applicable Conditions. Paper 1: [Abstract] Paper 2: [Abstract] Paper 3: [Abstract] … Please output in Markdown table format.

Format Conversion and Data Extraction

Convert messy data or text into the format you need — for example, extracting tabular data from paragraphs.

PROMPT EXAMPLE ▼
The following is a text description of experimental records. Please extract all numerical data and organize it into CSV format with columns: Sample ID, Measurement Temperature (K), Resistance (Ω), Notes. [Paste experimental record text here]

Writing Data Processing Scripts

Describe what you want to do and let AI write a Python script — then review and run it yourself.

PROMPT EXAMPLE ▼
Please write a Python script with the following functionality: 1. Read a CSV file (columns: time, voltage, current) 2. Calculate power at each time point (P = V × I) 3. Plot a line graph of time vs. power, with the maximum power point labeled 4. Save the figure as a PNG Please use pandas and matplotlib, and include clear comments.
⚠️ Before running, read through the script to verify the logic is correct — avoid overwriting raw data.

🎬 Claude Code Demo: Auto-Generate a Teaching Video

Paste this starter prompt into Claude Code — it will ask you one question at a time about the topic, slide count, and language, then automatically chain python-pptx, LibreOffice, edge-tts, and ffmpeg to produce a complete MP4 from scratch.

STARTER PROMPT ▼
I want to create a short educational video. First, create script.md to track everything. Ask me one question at a time and update script.md after each answer: 1. Topic / title of the video 2. Number of slides 3. Slide language (Chinese or English) 4. Voiceover language (Chinese or English) 5. Target audience 6. Auto-generate after script is ready, or wait for confirmation first? Tools to use: - python-pptx + LibreOffice -> PNG slides - edge-tts -> MP3 - ffprobe -> measure audio duration - ffmpeg -> assemble final MP4 Generation order (per slide): 1. edge-tts -> mp3 2. ffprobe -> get duration 3. ffmpeg -loop 1 -t {duration} -> clip.mp4 Then concat all clips into final MP4.
⚠️ Prerequisites: Claude Code, Node.js, Python, LibreOffice, python-pptx, edge-tts, ffmpeg
