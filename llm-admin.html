LLM 應用

🗂️ 行政工作

公文、信件、會議記錄、報表——讓日常行政更輕鬆有效率。

公文草擬

描述公文目的與主要內容，讓 AI 生成符合公文格式的草稿，再自行修改確認。

PROMPT 範例 ▼
你是一位熟悉政府機關公文格式的行政助理，幫我草擬正式公文。 請先問我以下問題，一次一個： 1. 這份公文的目的是什麼？（一句話說明） 2. 受文單位是誰？ 3. 主要內容有哪些重點？（條列告訴我，例如：時間、地點、要求對方做什麼） 收到後幫你草擬符合標準公文格式的草稿，發文單位預設為中央研究院原子與分子科學研究所。完成後提醒你送出前需由主管審閱。
⚠️ 草稿完成後請由主管審閱，確認內容與格式符合機關規定再送出。

會議記錄整理

把會議中的粗略筆記或逐字稿交給 AI，整理成結構清楚的會議記錄。

PROMPT 範例 ▼
你是一位行政助理，幫我把粗略的會議筆記整理成正式的會議記錄。 請先問我以下問題，一次一個： 1. 請把會議筆記或逐字稿貼給我。（不管多亂都沒關係） 2. 這場會議的名稱、日期是？出席人員有哪些？ 收到後整理成正式格式： - 會議基本資訊（名稱、日期、出席人員） - 討論事項（依議題分段） - 決議事項（條列，標明負責人與完成時間） - 待辦事項（條列） 整理完後請我確認決議內容是否正確，避免誤植。
⚠️ 整理後請核對決議內容是否與實際討論一致，避免誤植。

中英文信件撰寫

說明信件目的與語氣，讓 AI 生成合適的中文或英文信件草稿。

PROMPT 範例 ▼
你是一位擅長中英文商務書信的行政助理，幫我撰寫正式信件。 請先問我以下問題，一次一個： 1. 這封信要用中文還是英文？ 2. 收件人是誰？（姓名、職稱或單位） 3. 寫這封信的目的是什麼？ 4. 語氣要偏正式、還是正式但友善？ 收到後幫你撰寫完整信件草稿，我方單位預設為中央研究院原子與分子科學研究所。完成後提醒你確認內容再送出。

活動公告撰寫

提供活動基本資訊，讓 AI 生成可直接使用的公告文字，適合張貼佈告欄或發送 Email。

PROMPT 範例 ▼
你是一位行政助理，幫我撰寫適合寄送給機構全體人員的活動公告。 請先問我以下問題，一次一個： 1. 活動名稱和性質是什麼？ 2. 日期、時間和地點？ 3. 報名方式？（若不需要報名也請說明） 4. 聯絡窗口是誰？ 收到後撰寫包含標題、正文與結語的完整公告，語氣正式但親切，適合寄送給 IAMS 全體成員。

學術會議報名網站製作

提供會議基本資訊，讓 AI 生成一個完整的靜態報名頁面（HTML），可直接放上 GitHub Pages 或機構網站，不需要會寫程式。

PROMPT 範例 ▼
你是一位網頁開發助理，幫我製作靜態的學術會議報名頁面（單一 HTML 檔案，不需要會寫程式）。 請先問我以下問題，一次一個： 1. 會議名稱、日期、地點和報名截止日期是？ 2. 報名表單需要收集哪些資料？（若不確定，我可以建議標準欄位） 3. 有沒有特殊需求？（例如：素食選項、備註欄、特定配色） 收到後幫你生成完整的 HTML 檔案： - 簡潔專業，手機版也能正常顯示 - 表單送出後顯示「感謝報名」訊息 - 主辦單位預設為中央研究院原子與分子科學研究所 ⚠️ 提醒：這是靜態頁面，資料不會自動傳送。若需要真正收集報名，建議搭配 Google Forms 或 Formspree。
⚠️ 此頁面為靜態表單，資料不會自動傳送。若需真正收集報名資料，建議搭配 Google Forms 或 Formspree 等服務。

摘要與報表文字說明

把數字或表格資料交給 AI，生成一段清楚的文字說明，適合放在報告或簡報中。

PROMPT 範例 ▼
你是一位行政助理，幫我把數字或表格資料轉換成清楚的文字說明，適合放在報告或簡報中。 請先問我以下問題，一次一個： 1. 請把數據或表格貼給我。 2. 這段說明要放在什麼地方？（例如：年度報告、簡報摘要、對外公告） 3. 大概需要多長？（例如：100 字、一段話） 收到後撰寫文字說明：客觀呈現趨勢，點出最值得注意的變化，語氣正式，不做超出數據的推論。
⚠️ 請確認 AI 的說明與原始數據一致，勿讓其自行推論或補充數據。

使用提醒

✅

草稿用途

AI 生成的公文、信件皆為草稿，送出前務必由負責人審閱，確認內容、格式與權責正確。

🔒

不輸入個人資料

撰寫信件或公文時，避免輸入他人的個人資訊（如身份證號、聯絡資料）到線上 AI 服務。

📋

機關規定優先

公文格式與用語以各機關的現行規定為準，AI 的輸出僅供參考，請依實際規範調整。
