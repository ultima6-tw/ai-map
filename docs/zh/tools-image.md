# 🎨 影像生成 (AI Image Generation)

影像生成技術已進入「照片級真實」與「文字理解精準」的時代。

## 旗艦工具
- **[Midjourney](https://www.midjourney.com/)**: 影像美感之王，無論是構圖還是色調都具備極高的藝術感。
- **[Flux.1](https://blackforestlabs.ai/)**: 2024 年震撼業界的開源模型，解決了 AI 無法準確渲染文字與人手細節的痛點。
- **[DALL-E 3](https://openai.com/dall-e-3)**: 整合於 ChatGPT，最強大的語意理解能力，不需要複雜的 Prompt 就能產出正確的內容。

## 專業實戰技巧
- **控制與細節**: 使用 [Stable Diffusion](https://stability.ai/) 配合 **ControlNet**，可以精確控制人物姿勢、建築結構與線條。
- **影片生成**: 搭配 [Luma Dream Machine](https://lumalabs.ai/dream-machine) 或 [Runway Gen-3 Alpha](https://runwayml.com/gen-3-alpha)，可以將生成的靜態圖片轉為電影級動態影像。
