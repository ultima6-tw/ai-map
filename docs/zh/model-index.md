## 📐 模型階層說明 (Model Tiers)

主流模型通常分為不同階層，以應對不同的效能與成本需求。

### Google Gemini 3 系列
*   **Pro**: 旗艦型號，具備最強的推理與創作能力，適合複雜任務。
*   **Flash**: 高速度型號，在保持強大能力的同時提供極低的延遲。
*   **Flash-lite**: 極輕量型號，專為大規模簡單任務與極速反應設計。

### Anthropic Claude 系列
*   **Opus**: 最強大且最具創造力的模型，適合深度寫作與複雜邏輯。
*   **Sonnet**: 智慧與速度的平衡點，目前應用最廣泛的主流型號。
*   **Haiku**: 反應最快且成本最低的模型，適合簡單對話與快速檢索。

## 💰 訂閱制 vs. API 計費

選擇 AI 服務時，了解付費方式對於成本控管至關重要。

| 特性 | 月費訂閱 (Subscription) | API 連接 (Pay-as-you-go) |
| :--- | :--- | :--- |
| **對象** | 一般使用者、個人創作者 | 開發者、企業、自動化工作流 |
| **計費方式** | 固定月費 (約 $20 USD/月) | 按 Token 數量計費 (用多少付多少) |
| **介面** | 使用官方網頁或 App | 整合至第三方工具 (如 OpenClaw, TypingMind) |
| **隱私** | 資料可能被用於訓練 (除非手動關閉) | 資料通常預設不會被用於訓練模型 |
| **優點** | 使用成本預測性高，介面友善 | 極高靈活性，適合量大或自動化需求 |
