# AI 模型概覽 (AI Models Index)

本頁面整理當前主流的 AI 大語言模型，包含推理型、通用型以及開源權重模型。

!!! note "標記說明"
    *   🇨🇳：中國來源 (China-origin) 模型。
    *   🌐：美國/全球主流 (Global/USA) 模型。
    *   📖：開源權重 (Open Weights) 模型。

## 🧠 推理型模型 (Reasoning / Thinker)

專注於數學、程式碼及邏輯推理，通常具備「思維鏈 (Chain of Thought)」過程。

| 模型名稱 | 開發者 | 來源 | 類別 | 特點 |
| :--- | :--- | :---: | :---: | :--- |
| **OpenAI o3 / o3-mini** | OpenAI | 🌐 | 閉源 | 當前最強推理基準，具備深度思考能力。 |
| **DeepSeek-R1** | DeepSeek | 🇨🇳 | 📖 | 首個將 Reinforcement Learning 推向極致的開源推理模型。 |
| **Gemini 2.0 Thinker** | Google | 🌐 | 閉源 | 整合多模態與快速推理能力的思考型模型。 |
