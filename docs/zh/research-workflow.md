# 🔍 學術研究工作流 (AI-Assisted Research)

AI 不僅能摘要文字，更能輔助「發現隱藏關聯」與「驗證研究假設」。

## AI 輔助研究的核心流程
1. **問題探索 (Questioning)**: 
    - 使用 [Perplexity AI](https://www.perplexity.ai/) 或 [SearchGPT](https://openai.com/search) 進行初步文獻掃描。
2. **文獻審查 (Reviewing)**: 
    - 利用 [Elicit](https://elicit.org/) 或 [Consensus](https://consensus.app/) 從 2 億多篇學術論文中提取精確答案。
    - **[ResearchGPT](https://github.com/mshumer/researchgpt)**: 針對 PDF 庫進行精準提問與對話。
3. **數據分析與視覺化 (Analyzing)**: 
    - 使用 Python 代碼解釋器 (Code Interpreter) 處理複雜數據集。
4. **寫作與校對 (Writing & Proofing)**: 
    - 利用 AI 輔助論文草稿的邏輯梳理，並透過專用工具進行語法改進。

## 高階應用: AI 代理 (Agents)
- **多步驟代理**: 它可以幫你搜尋 50 篇論文，將其轉化為 Excel 表格，並總結出各篇論文的優缺點。
- **本地知識庫 (RAG)**: 將你的所有 PDF 論文放入 [QMD](./qmd.md) 或 [Nexus 知識圖譜](./agent.md)，實現全庫檢索。
