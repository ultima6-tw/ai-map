# 🖥️ 程式開發 (AI-Powered Coding)

AI 正在改變開發者的工作方式。從單純的程式碼補全到全自動化的開發代理。

## 熱門工具
- **[Cursor](https://cursor.com/)**: 目前最強大的 AI 代碼編輯器，基於 VS Code。提供強大的 `Composer` 模式，能跨文件編輯與修復代碼。
- **[GitHub Copilot](https://github.com/features/copilot)**: 最早且最穩定的 AI 助手，支援多種 IDE，並整合了 GitHub 的拉取請求 (PR) 審核。
- **[Claude Dev (Cline)](https://github.com/cline/cline)**: 一個開源的 VS Code 擴充功能，能將 Claude 3.5 Sonnet 變成你的開發代理，自動執行文件讀寫與命令列操作。
- **[Replit Agent](https://replit.com/ai)**: 適合快速原型開發，只需用文字描述，它就能幫你從零開始建立、部署完整的網路應用。

## 開發模型推薦
- **Claude 3.5 Sonnet**: 目前開發者公認的「邏輯之王」，在寫 Code 方面表現優異且極少出錯。
- **GPT-4o / o1**: OpenAI 的推理模型，適合處理複雜的系統架構與算法問題。
