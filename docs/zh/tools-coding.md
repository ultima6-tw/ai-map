# 🖥️ 程式開發 (AI-Powered Coding)

AI 正在改變開發者的工作方式。從單純的程式碼補全到全自動化的開發代理。

## 熱門工具
- **[Cursor](https://cursor.com/)**: 目前最強大的 AI 代碼編輯器，基於 VS Code。提供強大的 `Composer` 模式，能跨文件編輯與修復代碼。
- **[GitHub Copilot](https://github.com/features/copilot)**: 最早且最穩定的 AI 助手，支援多種 IDE，並整合了 GitHub 的拉取請求 (PR) 審核。
- **[Claude Dev (Cline)](https://github.com/cline/cline)**: 一個開源的 VS Code 擴充功能，能將 Claude 4.6 變成你的開發代理，自動執行文件讀寫與命令列操作。
- **[Claude Code](https://claude.ai/code)**: Anthropic 推出的官方命令列 AI 開發代理 (CLI Agent)。它能直接在終端機運作，深度整合開發者的 shell 環境，實現真正的 "Vibe Coding"。
    - **[OpenAI 內部實驗：Harness Engineering (YouTube 影片)](https://www.youtube.com/watch?v=ugKW_Dv37Bk)**：探討 OpenAI 如何透過自動化測試架構 (Harness) 讓 AI 自主修復 100 萬行程式碼，揭示未來 AI 工程化的核心轉向。
    - **[Claude Code 保姆級教學 (YouTube 影片)](https://www.youtube.com/watch?v=2pM-7fBXc_M)**：詳細介紹如何安裝與實戰使用 Claude Code，動動嘴就能做出任何應用。
    - **[Claude Code 零經驗實作教學 (YouTube 影片)](https://www.youtube.com/watch?v=XFmYkJJxsr8)**：教你如何在完全沒有經驗的情況下，使用 Claude Code 建立你的第一個應用。
    - **[Claude Code x OpenAI Codex Plugin：自動化審核與漏洞修補 (YouTube 影片)](https://www.youtube.com/watch?v=Kv0NHxIS33o)**：介紹如何整合 Claude Code 與 OpenAI 官方的 Codex Plugin，實現終端機內的自動 Code Review、漏洞攔截 (Review Gate) 與自動修復功能。
- **[Replit Agent](https://replit.com/ai)**: 適合快速原型開發，只需用文字描述，它就能幫你從零開始建立、部署完整的網路應用。

## 開發模型推薦
- **Claude 4.6**: 目前開發者公認的「邏輯之王」，在寫 Code 方面表現優異且極少出錯。
- **GPT-4o / o1**: OpenAI 的推理模型，適合處理複雜的系統架構與算法問題。
