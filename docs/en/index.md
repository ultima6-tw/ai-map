## 🛠️ Core Concepts

- **Local RAG (Retrieval-Augmented Generation)**: Privacy-first tech indexing private docs on local hardware (NAS/Workstations) for precise QA without cloud uploads.
- **Knowledge Graph**: Entity-relationship mapping beyond keyword search. Builds logical connections for deeper context and reasoning.
- **Digital Sovereignty**: Autonomy over data and compute. In the AI era, this includes developing models with local cultural traits (like TAIDE).
- **Neural Networks (ANN)**: The foundation of modern AI, mimicking brain neurons. Learns via parameters and gradient descent for tasks from classification to reasoning.
